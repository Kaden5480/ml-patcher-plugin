== MLPatcherPlugin ==
Find your game directory, this is most easily done
by going to the game in steam, pressing the settings for the
game (⚙️), selecting "Manage", then "Browse local files".

Copy/move the files in `Plugins` to `Plugins`.
Copy/move the files in `UserLibs` to `UserLibs`.
Install any other mods which depend on this plugin.
Start the game.
